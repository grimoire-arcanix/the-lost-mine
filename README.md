# the-lost-mine
A website to commemorate "The Lost Mine of Phandelver" run by me in January 2026

## Structure

- `index.html` — main chronicle / timeline
- `heroes.html` — player characters and allies
- `villains.html` — enemies and rivals
- `world.html` — locations, factions, and lore
- `gallery.html` — artwork, maps, and visual archive
- `style.css` — shared medieval parchment styling
- `script.js` — light scroll animation
